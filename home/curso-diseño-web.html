Curso de Diseño Web

Este curso está diseñado para equipar a los estudiantes con los conocimientos y habilidades necesarios para crear sitios web atractivos, funcionales y responsivos. A lo largo del curso, aprenderás los principios fundamentales del diseño web, las tecnologías clave utilizadas en la creación de sitios web y las mejores prácticas para optimizar la experiencia del usuario (UX).

Contenido del Curso

  • Introducción al Diseño Web: Historia, importancia y diferencias con el desarrollo web
  • HTML y CSS Básico: Estructura, etiquetas, selectores y propiedades
  • Diseño Responsivo: Media queries, frameworks CSS, diseño para móviles
  • Interactividad con JavaScript: Fundamentos, DOM, eventos, jQuery
  • Diseño UX/UI: Principios, wireframes, prototipos, herramientas de diseño
  • Optimización y SEO: Buenas prácticas, optimización de recursos, velocidad de carga, accesibilidad
  • Proyectos Prácticos: Creación de un portfolio, landing page, blog, proyecto final

Duración

El curso tiene una duración de 10 semanas, con clases impartidas dos veces por semana.

Precio

Este curso es completamente gratuito.

Requisitos

  • No se requieren conocimientos previos en diseño web
  • Una computadora con acceso a Internet
  • Software de edición de código (recomendamos VS Code)
